# ☕ Coffee Market Analysis Dashboard (Tableau)

## 📊 Project Overview

This project analyzes a dataset of 1,245 coffee brands reviewed between 2017 and 2022. The goal is to explore patterns in coffee roast types, global origins, roaster locations, and the relationship between price and rating using Tableau.

---

## 🎯 Objectives

* Analyze distribution of coffee roast types
* Identify top coffee-producing countries
* Examine roaster distribution globally
* Investigate the relationship between price and rating

---

## 📈 Key Insights

* There are 5 roast types, with Medium-Light dominating (~70%)
* Ethiopia and Brazil are the leading coffee origins
* The United States has the highest number of roasters (~62%), followed by Taiwan (~27%)
* There is a slight positive correlation between price and rating
